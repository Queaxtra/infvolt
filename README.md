
# infvolt

A site that fetch user information through Revol.chat

## How To Use


```bash
  npm install
  npm run dev
```
    
## Tools used on the site

- [revolt-api](https://www.npmjs.com/package/revolt-api)
- [Tailwindcss](https://tailwindcss.com/)
- [Svelte](https://svelte.dev/)
